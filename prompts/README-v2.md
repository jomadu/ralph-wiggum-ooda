# Minimal Composable Prompt Set v2

## Procedures Under Consideration

### Core Procedures

1. **bootstrap** - Create/update AGENTS.md operational guide
   - Default iterations: 1
   - Modifies code: No

2. **build** - Implement tasks from work tracking system
   - Default iterations: 5
   - Modifies code: Yes (only procedure that does)

### Draft Planning Procedures

3. **draft-plan-story-to-spec** - Converge plan for incorporating story into specs
   - Default iterations: 5
   - Modifies code: No
   - Writes to: Draft plan file (per AGENTS.md)

4. **draft-plan-bug-to-spec** - Converge plan for spec adjustments to drive bug fix
   - Default iterations: 3
   - Modifies code: No
   - Writes to: Draft plan file (per AGENTS.md)

5. **draft-plan-spec-to-impl** - Converge plan from gap analysis (specs → code)
   - Default iterations: 1 (could be increased for complex gaps)
   - Modifies code: No
   - Writes to: Draft plan file (per AGENTS.md)

6. **draft-plan-impl-to-spec** - Converge plan from gap analysis (code → specs)
   - Default iterations: 1 (could be increased for complex gaps)
   - Modifies code: No
   - Writes to: Draft plan file (per AGENTS.md)

7. **draft-plan-spec-refactor** - Converge plan from quality assessment of specs
   - Default iterations: 1
   - Modifies code: No
   - Writes to: Draft plan file (per AGENTS.md)

8. **draft-plan-impl-refactor** - Converge plan from quality assessment of code
   - Default iterations: 1
   - Modifies code: No
   - Writes to: Draft plan file (per AGENTS.md)

### Publishing Procedure

9. **publish-plan** - Publish converged draft plan to work tracking system
   - Default iterations: 1
   - Modifies code: No
   - Reads from: Draft plan file (per AGENTS.md)
   - Writes to: Work tracking system (per AGENTS.md)
   - Pushes to remote
